\doxysubsection{Detailed Description}
Represents an infinite double cone shape in the 3D scene. 

This class inherits from \doxylink{class_raytracer_1_1_a_shape}{AShape} and defines an infinitely long double cone (two cones meeting at their apex) with a center (apex), opening angle, axis normal, and color.
