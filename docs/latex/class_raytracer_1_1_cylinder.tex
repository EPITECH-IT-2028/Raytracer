\doxysubsection{Detailed Description}
Represents a cylinder shape in the 3D scene. 

This class inherits from \doxylink{class_raytracer_1_1_a_shape}{AShape} and defines a cylindrical object with a center, radius, height, normal vector (orientation), and color.
