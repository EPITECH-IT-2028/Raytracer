Computes the lighting contribution of this directional light at a given hit point. 

Calculates diffuse lighting based on the angle between the surface normal and the light direction. It also performs a shadow check by casting a ray from the hit point towards the light source. If an object is hit, the point is considered in shadow, and only a minimal light intensity is applied.
