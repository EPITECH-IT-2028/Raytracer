\doxysubsection{Detailed Description}
A composite shape that groups multiple \doxylink{class_raytracer_1_1_i_shape}{IShape} objects. 

This class inherits from \doxylink{class_raytracer_1_1_a_shape}{AShape} and allows treating a collection of shapes as a single entity. It delegates operations like ray intersection testing and transformations to its child shapes.
