\doxysubsection{Detailed Description}
Represents a camera in the 3D scene. 

The \doxylink{class_raytracer_1_1_camera}{Camera} class defines the viewpoint from which the scene is rendered. It manages properties like position (origin), orientation (yaw, pitch), field of view, and viewport dimensions. It provides methods to update its view vectors and generate rays for rendering.
