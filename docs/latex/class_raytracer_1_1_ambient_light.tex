\doxysubsection{Detailed Description}
Represents an ambient light source in the scene. 

This class inherits from \doxylink{class_raytracer_1_1_a_light}{ALight} and provides a constant light contribution to all objects in the scene, regardless of their position, orientation, or shadows. It simulates indirect lighting.
