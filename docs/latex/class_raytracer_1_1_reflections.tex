\doxysubsection{Detailed Description}
Represents a reflective material (e.\+g., mirror). 

This class implements perfect reflection. The color of the material is determined by tracing a new ray in the reflection direction and blending its result with the material\textquotesingle{}s base color.

Computes the color for a reflective material. 

Calculates the reflection direction based on the surface normal and view direction. A new ray is traced in this reflection direction, and its color is blended with the material\textquotesingle{}s base color using a fixed reflectivity factor.
