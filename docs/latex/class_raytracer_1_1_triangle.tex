\doxysubsection{Detailed Description}
Represents a triangle shape in the 3D scene. 

This class inherits from \doxylink{class_raytracer_1_1_a_shape}{AShape} and defines a triangular object specified by three vertices (p1, p2, p3), a normal vector, and a color.
