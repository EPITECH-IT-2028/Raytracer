\doxysubsection{Detailed Description}
Abstract base class for all shapes in the \doxylink{namespace_raytracer}{Raytracer}. 

This class implements the \doxylink{class_raytracer_1_1_i_shape}{IShape} interface and provides common properties and methods for shapes, such as center, color, material, shininess, and translation.
