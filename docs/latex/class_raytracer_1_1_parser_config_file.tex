\doxysubsection{Detailed Description}
Parses configuration files for the \doxylink{namespace_raytracer}{Raytracer} scene. 

This class uses libconfig++ to read and interpret .cfg files, populating \doxylink{class_raytracer_1_1_camera}{Camera}, \doxylink{class_raytracer_1_1_shape_composite}{Shape\+Composite}, and \doxylink{class_raytracer_1_1_light_composite}{Light\+Composite} objects with the data found in the configuration. It supports parsing camera settings, primitives (spheres, cylinders, planes, etc.), lights (ambient, point, directional), and importing other scenes. It also handles parsing .obj files for object primitives.
