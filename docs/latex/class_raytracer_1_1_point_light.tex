\doxysubsection{Detailed Description}
Represents a point light source in the scene. 

This class inherits from \doxylink{class_raytracer_1_1_a_light}{ALight} and provides specific implementation for a light source that emanates light in all directions from a single point in space. It contributes to diffuse and specular lighting and can cast shadows.
