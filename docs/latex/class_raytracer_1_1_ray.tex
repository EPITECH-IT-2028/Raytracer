\doxysubsection{Detailed Description}
Represents a ray in 3D space, defined by an origin and a direction. 

A ray is typically used for ray casting and ray tracing algorithms to determine intersections with objects in a scene. The ray is parameterized as P(t) = origin + t \texorpdfstring{$\ast$}{*} direction.
