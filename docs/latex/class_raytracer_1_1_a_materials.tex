\doxysubsection{Detailed Description}
Abstract base class for materials. 

This class implements the \doxylink{class_raytracer_1_1_i_materials}{IMaterials} interface and serves as a base for concrete material implementations. It provides default constructor and destructor. The {\ttfamily compute\+Material} method remains pure virtual, requiring derived classes to implement their specific shading logic.
