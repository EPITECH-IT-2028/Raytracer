\doxysubsection{Detailed Description}
Abstract base class for all light sources in the \doxylink{namespace_raytracer}{Raytracer}. 

This class implements the \doxylink{class_raytracer_1_1_i_light}{ILight} interface and provides common properties and methods for lights, such as intensity, color, direction (for directional lights), and type.
