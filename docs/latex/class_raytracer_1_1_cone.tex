\doxysubsection{Detailed Description}
Represents a finite cone shape in the 3D scene. 

This class inherits from \doxylink{class_raytracer_1_1_a_shape}{AShape} and defines a cone with a circular base, an apex, specified by its center (base center), radius (of base), height, axis normal, and color.
