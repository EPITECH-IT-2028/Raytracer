Detailed Description

Represents a reflective material (e.g., mirror).

This class implements perfect reflection. The color of the material is determined by tracing a new ray in the reflection direction and blending its result with the material's base color.

Computes the color for a reflective material.

Calculates the reflection direction based on the surface normal and view direction. A new ray is traced in this reflection direction, and its color is blended with the material's base color using a fixed reflectivity factor.

Parameters
normalThe normal vector at the hit point.
viewDirThe direction from the hit point towards the camera.
hitPointThe point of intersection on the surface.
colorThe base color of the reflective material.
shapesThe composite of all shapes in the scene.
lightsThe composite of all lights in the scene.
cameraThe scene camera.
depthThe current recursion depth for ray tracing.
rayColorFuncA callback function to compute the color of a new ray.
Returns
Math::Vector3D The final computed color, blending the material's color with the reflected color.

If the recursion depth is not exhausted, it calculates the reflection direction and casts a new ray from the hit point (slightly offset) in that direction. The color obtained from this reflected ray is then blended with the object's own color using a fixed reflectivity factor (0.9 for reflected, 0.1 for base color).
