Detailed Description

Represents a 3D rectangle defined by an origin and two side vectors.

This class can be used to define a rectangular area in 3D space, for example, as a screen plane for a camera or a simple rectangular light source.

Default constructor for Rectangle3D. Initializes a 2x2 rectangle in the XY plane, centered around (-0,-0,-1) if origin is (-1,-1,-1). More precisely, it creates a rectangle from (-1,-1,-1) to (1,1,-1) if left_side was (0,2,0) and bottom_side was (2,0,0). The default values define a rectangle from P0=(-1,-1,-1) (origin) to P1=P0+bottom_side = (1,-1,-1) to P2=P0+left_side = (-1,1,-1) to P3=P0+bottom_side+left_side = (1,1,-1).

The point is determined by parametric coordinates u and v, which scale the _bottom_side and _left_side vectors respectively. P(u,v) = origin + u * bottom_side + v * left_side.
