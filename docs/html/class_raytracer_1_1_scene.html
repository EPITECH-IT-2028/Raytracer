Detailed Description

Manages the overall scene, rendering, and user interaction.

This class is responsible for initializing the scene from a configuration file, handling user input for camera control, managing the rendering loop, and displaying the rendered image. It also handles plugin loading.

void Raytracer::Scene::render ( )

Renders the scene.

Renders the scene and handles the main event loop. Initializes the renderer, renders an initial frame, creates a PPM file, and then enters the main loop. In the loop, it polls for events, handles input, updates camera quality, re-renders the scene, updates the image, and displays it.
