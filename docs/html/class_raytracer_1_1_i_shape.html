Detailed Description

Interface for all drawable shapes in the Raytracer.

Defines the common contract for shapes, including methods for ray intersection testing, normal calculation, transformations, and accessing properties like center, color, and material.

◆ hits()

virtual std::tuple< double, Math::Vector3D, const IShape * > Raytracer::IShape::hits ( const Raytracer::Ray & ray) const
pure virtual

Calculates the intersection of a ray with the shape.

Parameters
rayThe ray to test for intersection.
Returns
A tuple containing:
  • double: The distance from the ray's origin to the intersection point (t).
  • Math::Vector3D: The color of the shape at the intersection point.
  • const IShape*: A pointer to the shape itself if an intersection occurs. Implementations should return a t value <= 0 or a null shape pointer if no hit.

Implemented in Raytracer::AShape, Raytracer::Cone, Raytracer::ConeInf, Raytracer::Cylinder, Raytracer::CylinderInf, Raytracer::Object, Raytracer::Plane, Raytracer::ShapeComposite, Raytracer::Sphere, and Raytracer::Triangle.
